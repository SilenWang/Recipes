# Recipes

This repository contains recipes for my personal channel on `prefix.dev`: [https://prefix.dev/channels/sylens](https://prefix.dev/channels/sylens).

It hosts software that I need for work but is unavailable through other public channels.

Use [pixi](https://pixi.prefix.dev/latest/) and [rattler-build](https://rattler-build.prefix.dev/latest/) to build packages.
